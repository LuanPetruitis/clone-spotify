# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/LuanPetruitis/clone-spotify.git

# entrar na pasta do projeto front end web
cd clone-spotify

# instalar dependências
npm install

# executar o projeto
Executar a extensão live-server
```
